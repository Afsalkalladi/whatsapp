### 🎯 QUICK START (5 COMMANDS)

1. cd /tmp/whatsapp_cv_manager
2. chmod +x setup.sh && ./setup.sh
3. cp .env.example .env  # Then edit with your API keys
4. python3 check_setup.py
5. make run  # In another terminal: make ngrok

Configure WhatsApp webhook → Done! 🚀
